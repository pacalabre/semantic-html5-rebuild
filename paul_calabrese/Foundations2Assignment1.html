Web UI Accelerator

Overview

In this class, students will learn advanced HTML5, CSS3, Sass, responsive web design, and JavaScript to be able to integrate with development teams. This course also covers building a simple server-side app leveraging concepts from Node.js or Ruby on Rails, and concepts around iterative design development to include style tiles, sketching, wireframing, and algorithmic flow diagrams.

Prerequisites

The Web UI Development Accelerator is intended for those with a background in design (graphic design, UX Design, web design, etc.) who would like to supplement their already marketable skills with web development training. A design portfolio that showcases student work is required for all Web UI Development courses.This course requires a fundamental understanding of computer science and web development equivalent to the topics taught in our Foundations I and Foundations II: Web UI Development courses.In addition, we require completion of Unix & Git for Everyone prior to beginning this course. Students who are already experienced with Unix, Git, and the command line may request to test out of this workshop.

Prework

Students will need to complete approximately 20 hours of prework prior to beginning this course, including:

  • Unix and Git for Everyone
  • HTML & CSS tutorials on Codeacademy
  • Familiarity with package management systems
  • Familiarity with the following development environments:
    • RVM
    • Middleman
    • Node.js

    Course Curriculum

    Development Concepts

    • Building simple back-end server environments such as Node.js, Ruby on Rails, and Python
    • Agile development best practices
    • Reading and writing APIs
    • Development, build, and deployment strategies
    • Building prototype projects using front-end frameworks

    Advanced HTML

    • Semantic structure using HTML5
    • Advanced architectural and modular HTML development concepts
    • HTML5 form development
    • Abstractions of HTML, such as Haml, Jade, and Markdown

    Advanced CSS

    • Advanced CSS layout techniques
    • The impact of inheritance, specification, and writing efficient selectors
    • Advanced architectural and modular CSS development concepts
    • SMACSS and BEM
    • Presentational code
    • Living style guides
    • Proper UI testing

    Sass

    • Core basics
    • Advanced Sass techniques for building modular UI code
    • Scalable and maintainable UI code bases

    UI Role in Front-End Development

    • Iterative design development
      • Style tiles
      • Sketching
      • Wireframing
      • Algorithmic flow diagrams
    • Building and implementing a workflow driven by a style guide
    • The role of UX principles in front-end development
    • Responsive web design

    Homework Policy

    Code Fellows courses and assignments are graded on a complete/incomplete system. Students must complete 90% of assignments to complete the course. Assignments are on a point system of 10 points. Ten points will be awarded for completing the assignment. There are no partial grades.Co-working and Q&A class time are the primary feedback opportunities. Students are expected to take advantage of feedback and the one-on-one, in-person time for code reviews by instructors and their peers.Students are asked to keep the instructors up to date on their progress throughout the course. Make-up assignments and grace periods for late work may be allowed based on instructor discretion.

    Attendance Policy

    Punctuality, participation in discussions, completion of assignments, and demonstration of professional courtesy to others are required. Attendance will be taken at the beginning of every class. Passing requires at least 75% attendance. Students should always contact the instructors ahead of time if unable to attend all or part of a class session.

    Job-Offer Guarantee

    We believe that students who successfully graduate from Development Accelerators are ready to make immediate, meaningful contributions to a software development team. If a graduate does not receive a job offer within nine months of graduating, we will refund their tuition.The guarantee applies only to Development Accelerator students who graduate in good standing and actively and professionally conduct their job search in the interim between graduation and a job offer. But we are here to help at each step along the way.
